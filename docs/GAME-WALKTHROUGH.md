## 一、开始游戏

1. 在项目根目录运行 `npm run serve`
2. 浏览器打开 `http://localhost:5173/`
3. 进入「静园」官网首页

**第一眼你会看到什么:**

- 三张"学员故事"卡片,其中第一张是「林 · 27 · 上海」
- 页脚附近的"已服务 14,200,003 名学员 · 重生成功率 99.7%"

## 二、第一幕 · 温柔的假象

  你看到这三处,就懂了。

从 admin 页面底部(或直接 URL 访问)可以进入三个结局。

- "第 48 号样本已锁定。"

### 结局 C:`/endings/hidden.html` · **隐藏结局(硬核)**

## 七、崩坏档级(全程发生)

游戏有一个**全局腐化度**变量 `localStorage.serene.corruption`,范围 [0,3]。

| 触发点 | 腐化度 | 视觉 |
|---|---|---|
| 第一次进入 | 0 | 全部页面正常 |
| 进入 `/blog/` | +1 → 1 | 档 1 温柔渗血:细微 CSS 异常(whisper 闪烁、计数闪烁) |
| 进入 `/admin/` | +1 → 2 | 档 2 界面撕裂:后台页面变黑底蓝字 + 扫描线 |
| 已访 ≥ 2 后访问 `/endings/hidden.html` | 不增加 | 档 3 凝视:全站黑底粉字 + 眼睛元素 |

—— **档 3 不会自动全局切换**,只在你访问 `/endings/hidden.html` 时短暂出现;如果你想全程体验档 3 凝视感,可以在 DevTools Console 里手动:
```js
localStorage.setItem('serene.corruption', '3'); location.reload();
```

## 十、给 ARG 老炮的硬核版

- **多刷几次**,注意每次进入页面时,有些数字(47,812)和文案会有微妙变化(`corruption.js` 文案池随机抽取)
- **看源码**:每个 HTML 的注释都可能藏着隐藏信息(`<!-- ... -->` 在 HTML 里、`//` 在 JS 里)
- **看 favicon / robots.txt / 404 页**:这些是 ARG 经典藏彩蛋的地方
- **DevTools Console**:手动改 `localStorage` 可以跳过任何门禁
  ```js
  // 强行进入所有站:
  for (const id of ['serene-home','blog','forum','wechat','social','endings']) {
    // createState() 会自动写入默认值
  }
  // 强行档 3:
  localStorage.setItem('serene.corruption','3');
  location.reload();
  // 重置一切重来:
  localStorage.clear(); location.reload();
  ```

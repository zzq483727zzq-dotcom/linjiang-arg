## 十二、细思极恐的隐藏细节

### 数字 47 反复出现
- 首页"今天已有 47,812 人"
- 博文 1:林"过去 90 天有 47 次想结束"
- 博文 2:"IQGG 这个代号出现了 47 次"
- 论坛:"47 条未接来电" + 号码 400-88-0047
- admin 样本编号:0471-0475
- 朋友圈最后一条:"您访问过的所有页面累计已被记录 47 次"
- 404 页:"你在这个站点被访问了 48 次"(48 = 47 + 1)
- 隐藏结局:"你是第 48 张脸"

—— **47 不是巧合,这是一个循环实验;你是第 48 号样本**。

### 客服对话最后一句
- "您是做什么工作的? 这种细节您是从哪里听到的? ……请把这段对话关掉。这也是为您好。"
- —— **林(用户)已经不是用户了,而是被标记的"样本"**

### admin 的 NPS 涂黑
- 涂黑 `98.4%` —— 99.7% 是首页宣传的"重生成功率",98.4% 是后台的真实数字。**对外宣传比真实数字高**。

## 十三、给非技术玩家的话

如果中途卡住:
1. 用鼠标选中 admin 页面里那些黑色方块,会显示出被涂掉的原文
2. 用 DevTools Console 运行 `localStorage.setItem('serene.corruption', '3'); location.reload();` 可以强制档 3 视觉
3. 整个游戏没有时间限制,没有"死路",所有线索都在页面文本里

**强烈建议:** 第一遍不要看这个攻略,纯盲玩一遍,看看自己漏了什么。然后再翻一遍——你会有不同的体验。

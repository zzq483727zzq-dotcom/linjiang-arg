# 临江 · 幸福家园 ARG — 官方主线入口

> 旧「静园」folk 路径（`/blog` `/forum` `/chat` `/social` `/admin` `/endings` `/contact` 等）**不在主线内**。
> 若仍打开旧 URL，页面会提示档案已迁移并链回 `/`。

## 官方入口

| 路径 | 用途 |
|------|------|
| `/` | 说明页（intro）→ 开始调查 |
| `/phone/` | 林屿手机桌面与各 App |
| `/city/` | 临江政府公开站 / 新闻 / OA |
| `/home/` | 幸福家园公开站 |
| `/hongke/` | 弘科生物 |
| `/bar/` | 临江吧 |
| `/staff/` | 家园员工端 |
| `/ending/` | 终局页（单数 `ending`，非旧 `endings`） |

## 主路径（摘要）

1. `/` 填名 → `/phone/`
2. 微信 / 邮件 / 浏览器 → 政府站与评论
3. 临江吧 + 弘科线索 → OA 门禁
4. 幸福家园公开站 → 员工端
5. 爆点 / 红字墙 → `/ending/*`
